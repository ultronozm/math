Oleksiy Klurman, On Elliott’s conjecture and applications

Let’s talk about some recent progress on some topic of multiplicative number theory, starting with two applications of this recent progress.

One application is the following:

  1. Take any binary sequence \(\{a_n\}_{n \in \mathbb{N} } = \{\pm 1\}^{\mathbb{N} }\). Suppose you measure the discrepancy of this sequence along arithmetic progressions. Then Erdos conjectured that \[\sup_{n, d} \left\lvert \sum_{k = 1}^n a_{k d} \right\rvert = + \infty.\] This was eventually proved by Tao (2015), following a partially successful polymath project.

  2. The second application concerns (completely) multiplicative functions \(f\), thus \(f (m n) = f (m) f (n)\), that are \(1\)-bounded int he sense that \(\lvert f \rvert \leq 1\). Suppose for instance that \(f : \mathbb{N} \rightarrow \{\pm 1\}\). Then the problem is to determine which multiplicative functions avoid some sign patterns.

    1. Clearly there are no multiplicative functions that take the value \(-1\) from some point on. In other words, there are none that have no \(+1\) at all from some point on.

    2. But what about functions that never take two values \(+1\) in a row from some point on? Consider \[x^2 - p q y^2 = 1, \quad f (p) = f (q ) = - 1.\] Then \[f (p q y^2 ) = f (q ) f (p) f (y)^2 = 1,\] \[f (p q y^2 + 1) = f (x^2 ) = 1.\]

    3. Lehmer conjectured, and Schur proved using a combinatorially argument, that there are exactly two functions, completely multiplicative, that avoid three consecutive values \(+1\), or equivalently, three quadratic residues in a row. \[\chi_3^{\pm} (p) = \begin{cases} 1 & \text{ if } p \equiv 1(3), \\ -1 & \text{ if } p \equiv -1(3), \\ \pm 1 & \text{ if } p = 3. \end{cases}\] (In the 80’s, Hildebrand asked the same question about more general sign patterns, and these questions have been understood.)

    4. Four in a row is a much more difficult question.

      Conjecture 1 (Hudson, 1974; Mills, 1965). There are thirteen functions, in addition to the two mentioned above, that avoid \(++++\). They are \(\chi_p^{\pm}\) with modulus \(p \in \{5, 7, 11, 13, 53\}\). There are two functions that look like characters mod \(4\): \[g_1 (p) = g_2 (p) = (-1)^{\frac{p-1}{2}}, \quad p \neq 2.\] \[g_1 (2) = - g_2 (2) = - 1.\] \[g_3 (p) = 1, p \neq 2.\] \[g_3 (2) = - 1.\]

Theorem 1 (KMV, 2023). Conjecture 1 is true.

Let’s begin with some thoughts. Let’s try to understand \[\frac{1}{16} \sum_{n \leq N} (1 + f (n )) (1 + f (n + 1)) (1 + f (n + 2 )) (1 + f (n + 3)).\] If we open everything up, then we get a main term plus a \(k\)-point correlation sum for each \(k \in \{1,2,3,4\}\). \[\frac{N}{16} + \sum f (n) f (n + 1) f (n + 2) f (n + 3) + \sum (\text{3 pt}) + \sum (\text{2 pt}) + \sum_{n \leq N} f (n) + \dotsb.\] We aim to bound everything but the first term.

Question 1. When is \(\sum_{n \leq N} f (n) \gg N\)? We recall here that \(f : \mathbb{N} \rightarrow \{\pm 1\}\) is completely multiplicative, and takes values in \([-1, 1]\).

Wirsing says that no matter which multiplicative function one takes with the above conditions, one has some cancellation, i.e., \[\sum_{n \leq N} f (n ) = o (N),\] unless \[\sum \frac{1 - f (p)}{p} < \infty,\] or equivalently, \[\sum_{f (p) = - 1} \frac{1}{ p} < \infty.\] In other words, \(f\) is close to \(1\) in a certain sense on the primes. In that case, the mean value is \[\mathcal{M}_N \sim N \prod \left( 1 - \frac{1}{p} \right) \left( 1 + \frac{f (p)}{p} + \dotsb \right).\]

How about the two and three point correlations? Chowla said that the correlations of Liouville or Mobius always vanish, except when they obviously don’t. Here we recall that \(\lambda (n) = (- 1 )^{\Omega (n)}\).

Conjecture 2 (Chowla, 1964). For all distinct \(h_1, \dotsc, h_k \in \mathbb{N}\), we have \[\sum_{n \leq N} \lambda (n + h_1 ) \lambda (n + h_2 ) \dotsb \lambda (n + h_k ) = o(N).\]

Conjecture 3 (Elliott, 1994). If \(f : \mathbb{N} \rightarrow \mathcal{U}\) is a \(1\)-bounded completely multiplicative complex-valued function, then its correlations vanish \[\sum_{n \leq N} f (n + h_1 ) \dotsb f (n + h_k ) = o(N)\] unless \(f(n)\) is “close” to \(n^{i t} \chi (n)\) in the sense that for some \(t\) and \(\chi\), we have \[\mathbb{D} (f, n^{i t } \chi, \infty ) < \infty,\] with notation as below.

Definition 2. We define the squared distance between two multiplicative functions at scale \(x\) to be \[\mathbb{D}^2 (f , n^{i t } \chi (n), x) = \sum_{p \leq x} \frac{1 - \Re (f (p) \overline{\chi (p)} p^{- i t})}{p}.\]

Remark 3. Consider functions \(f\) such that \[\mathbb{D} (f, n^{i t } \chi, \infty ) = \infty\] for all \(t\) and \(\chi\). Ergodic theorists call such functions aperiodic.

Remark 4. The largest that the distance \(\mathbb{D}^2\) can be is \[2 \sum_{p \leq x} \frac{1}{p} \sim 2 \log \log x.\]

Matomaki, Radziwill and Tao pointed out that this conjecture is wrong. The class of functions for which it is wrong is now called the MRT class. The idea is to construct a function \(f\) that looks like different \(n^{it } \chi\)’s at different scales, but looks like none of them in particular at all scales. In more detail, we are trying to understand \[\label{eqn:d1bbefb37a02}\tag{1} \sum f (n) \overline{f (n + 1)}.\] At scale \(p \leq S_m\), we take \(f (p) \sim p^{i t_m }\). Then \((1)\) is large. At the next scale \(S_{m+1}\), which we take to be much, much larger than \(S_m\), we take a different character. Then \(\mathbb{D} (f, n^{i t_m }, s_m )\) is small, but for all \(t\) and \(\chi\), we have \(\mathbb{D} (f, n^{i t } \chi, \infty ) = \infty\). It’s key here that this scale be much much larger than \(s_m\).

Theorem 5 (Tao, 2015; Terivainen–Tao, 2018). (Tao proved something about \(2\)-point correlations, which was generalized with Terivainen to odd-point correlations.) Assume one of the following conditions:

  1. \(k = 2\), \(f\) is strongly aperiodic: \[\inf_{ \substack{ \lvert t \rvert \leq A x \\ q \leq (\log x )^A } } \mathbb{D} (f, n^{i t } \chi, x) \xrightarrow{x \rightarrow \infty } \infty.\] Then there exists \(\mathcal{X} \subseteq \mathbb{N}\) with density one, i.e., with \[\lim_{X \rightarrow \infty } \frac{\sum_{ \substack{ n \in \mathcal{X} \\ n \leq x } } 1 }{ \sum_{n \leq x} \frac{1}{ n} } = 1,\] such that for all \(h_1, h_2\) distinct, we have \[\lim_{ \substack{ x \rightarrow \infty \\ x \in \mathcal{X} } } \frac{1}{x} \sum_{ n \leq x } f (n + h_1 ) \overline{f (n + h_2 )} = 0.\]

  2. \(k\): odd. Suppose for simplicity that \(k = 3\) and \(f : \mathbb{N} \rightarrow [-1, 1]\) and that \(f\) is super strongly aperiodic: for each \(\chi\), \[\mathbb{D}^2 (f, \chi, x) \gg \log \log x.\] (We can ignore \(t\)’s because \(f\) is real.) Then the same conclusion as in the previous case is true.

Theorem 6 (KMT, 2023). Same conditions as above:

  1. For \(k = 2\), and \(f\) aperiodic (rather than strongly aperiodic), the same conclusion holds as in Theorem 5, except that we can only take \(\mathcal{X}\) to have upper logarithmic density: same condition as above, but using \(\limsup\) rather than \(\lim\). (This is necessary; one can otherwise construct counterexamples.)

  2. For \(k\) odd, with \(f\) aperiodic, and with upper logarithmic-logarithmic density.

Nina Zubrilina, Root Number Correlation Bias of Fourier Coefficients of Modular Form

Statistical properties of Fourier coefficients of automorphic forms. Distribution of \(a_\pi (p)\). Let’s backtrack a bit and give some informal overarching context for the more refined phenomena that we’ll discuss later.

We start with the conjecture of Sato and Tate, which describes what we’ll call the horizontal distribution. This refers to the behavior of Fourier coefficients for a fixed arithmetic \(L\)-function. To its coefficients \(a_L (p)\), we associate a measure \(\mu\), given by the Haar measure on a certain group \(G_L\).

Example 7. Given a non-CM elliptic curve \(E/\mathbb{Q}\), we have \[a_p = \sqrt{p} \lambda (p) = 2 \sqrt{p} \cos \theta_E (p).\] Then as \(p \rightarrow \infty\), the \(\theta_E (p)\) follow the famous Sato–Tate distribution: \[\mu (\theta) = \frac{2}{ \pi } \sin^2 \theta \, d \theta .\]

An orthogonal picture to this is what we’ll refer to as the vertical distribution. The way we’ll like to think of it is that we have some packet \(\mathcal{F}_N\) of automorphic forms \(\pi\), and we’d like to prescribe that the conductors of the elements of these packets have roughly the same size: \[\frac{N (\pi) }{N} = 1 + o(1).\] One example of this would be to take a Hecke basis of modular forms of some fixed weight and level \(N\): \[\mathcal{F}_N := \Gamma_0(N,k).\] Then the Satake parameters \[\left\{ \theta_\pi (p) \right\} \approx_N \mu_p := \text{$p$-adic Plancherel}.\] We won’t write it down, but remark that it approaches the Sato–Tate distribution as \(p \rightarrow \infty\). This is a theorem of Serre.

How about the expectation? Let’s write it as the correlation: \[\operatorname{Cor} (\mathcal{F}_N, p) := \mathbb{E}_{\pi \in \mathcal{F}_N } a_\pi (p) = \frac{1}{ \lvert \mathcal{F}_N \rvert} \sum a_\pi (p).\] In this vertical averaging, this expectation, in many many cases, will be zero. One can heuristically derive this expectation from the fact that these distributions are symmetric. See the paper of Sarnak–Shin–Templier.

A similar question can be asked back on the horizontal size. If we consider a packet of primes \[P_X = \left\{ p : p = (1 + o(1)) X \right\},\] then we can again ask about this correlation function \[\operatorname{Cor} (P_X , \pi ) \sqrt{X} = \text{almost periodic function of } \log X.\] This will oscillate about the mean. It’s a generalization of the Chebyshev bias. A reference is a paper of Rubinstein–Sarnak.

What we’ll discuss today is a hybrid phenomenon, where we consider a double correlation \[\operatorname{Cor} (P_X, \mathcal{F}_N ) = \mathbb{E}_{ \substack{ p \in P_X \\ \pi \in \mathcal{F}_N } } a_\pi (p).\] We’ll take \(N\) and \(X\) to depend upon each other in some way.

The problem that we’ll talk about comes from the world of computer science, historically speaking. It comes from a machine learning paper of He, Lee, Oliver and Pozdnyakov. They fed a bunch of invariants coming from number theory into a machine learning algorithm and tried to guess others. For instance, they considered a family \[\mathcal{E} = \mathcal{E}_{[N _1, N_2]}^k = \left\{ E / \mathbb{Q} : \mathop{\mathrm{rank}}(E) = r, \, N(E) \in [N_1, N_2] \right\}.\] They fed in Frobenius traces for small primes and tried to guess the rank. They observed the following curious feature: \[\frac{1}{\lvert \mathcal{E} \rvert} \sum_{E \in \mathcal{E} } a_E (p) = \operatorname{Cor} (\mathcal{E}, p) = \text{``interesting''}.\] This is best illustrated by a picture.

Picture shows interval between 7000 and 10000. \(x\)-axis is the parameter \(p\). Blue line: elliptic curves of rank zero. Red line: elliptic curves of rank one. There’s this weird oscillation behavior.

Bottom picture: blue is rank zero, green is rank two. Here it looks sort of like the same picture, just shifted a bit.

This might have gone completely unnoticed, but Sutherland stumbled upon it and performed a bunch of computations. He mentioned a couple things about this phenomenon. You can think about it not as a phenomenon about the rank, but rather about the root number (noting the role played by parity). The second thing he noticed is that perhaps the correct intervals to take are “geometric intervals”: for \(w \in \{\pm 1\}\), \[\mathcal{E}_M^w = \left\{ E / \mathbb{Q} : w(E) = w, \, N(E) \in \begin{cases} [M,2 M] \\ [M, (1 + o(1)) M]. \end{cases} \right\}\] That is to say, one can consider either a dyadic interval or a short interval. What Sutherland noticed is that when you switch to these geometric intervals, there’s a funny phenomena that shows up, that we’ll call scale invariance.

The picture shows two intervals at unrelated scales (dyadic case) for which the plots have similar shapes, suggesting that perhaps \[\operatorname{Cor} (\mathcal{E}_M^w , p ) \approx w \mathcal{M} \left( \frac{p}{M} \right).\]

We observe next that the ordering is very important. If we plot instead by Faltings height, naive height, \(j\)-invariant, then we just get complete gibberish. It matters that we order by conductor.

The last observation is that, rather than just elliptic curves, we can consider broader families \[\mathcal{F}_M^w = \left\{ \text{weight two newforms for } \Gamma_0 (N) : N \in I_M, \text{degree $d$} \right\}.\] The picture becomes very pretty if we impose no condition on degrees. The theorem we’ll present today is that one can actually write down this function and prove something about it. We give an explicit function. The short intervals are nicer to work with.

Some more pictures. Nice.

Let’s write one line about the proof. It’s basically that \[\sum_{p} a_f (p) w (f) = \pm \mathop{\mathrm{trace}}(T (p) \circ W_N ).\] Apply the Selberg trace formula. Unravel it. Get that the above is basically the sum of a bunch of class numbers, depending upon \(p\) and \(N\). We apply a theorem of Skoruppa–Zagier. Get that the trace is a sum \[\sum_{\lvert k \rvert \leq 2 \sqrt{P / N}} H_1 (k^2 N^2 - 4 P N) - (P + 1).\]

Let’s add a few words about how this ties up with the Katz–Sarnak philosophy. One way to think about that philosophy is that if we take a family \(\mathcal{F}\) with symmetry type \(\operatorname{O}(\infty)\), then the \(1\)-level density conjecture says that if we take the parameter \(X\) for the prime to be the conductor to some power, say \(N^a\), and we compute this correlation, then as \(N \rightarrow \infty\), that correlation should behave as follows: \[\operatorname{Cor} (\mathcal{F}_N^w , P_X ) \xrightarrow{N \rightarrow \infty } \begin{cases} 0 & \text{ if } a < 1, \\ - \frac{w}{2} & a > 1. \end{cases}\] Since we took \(p\) to be of size \(N\), the murmurations occur at the phase transition between the gap in asymptotics between the two cases of the \(1\)-level density conjecture.

Yu-Chen Sun, On divisor bounded multiplicative functions in short intervals

Introduction

Huxley, 1972: prime number theorem in short intervals: for any \(H \geq X^{7/12 + \varepsilon}\), we have \[\sum_{X < n \leq X + H } \Lambda (n ) = (1 + o(1)) H.\] Similarly, we have \[\sum_{X < n \leq X + H } \mu (n) = o(H)\] and \[\sum_{X < n \leq X + H } \lambda (n) = o(H).\] Matomaki and Teravainen recently showed that in the Mobius and Liouville cases, the results hold for all \(H \geq X^{0.55 + \varepsilon}\), but they cannot prove this for the von Mangoldt function (their arguments are based upon multiplicative properties). Under RH, the above results hold for all \(H \geq X^{1/2 + \varepsilon}\).

In 2016, Matomaki and Radziwill proved the following breakthrough result:

Theorem 8 (Matomaki–Radziwill). Let \(f : \mathbb{N} \rightarrow [-1,1]\) be a multiplicative function. For almost all \(x \in [X, 2 X]\), we have \[\frac{1}{h} \sum_{x < n \leq x + h} f (n) - \frac{1}{X} \sum_{X < n \leq 2 X} f (n) = o(1)\] provided that \(h \rightarrow \infty\) as \(X \rightarrow \infty\).

  • For any \(\varepsilon> 0\), the interval \([X, X + C (\varepsilon) \sqrt{X }]\) contains \(X^\varepsilon\) numbers.

  • For every integer \(h \geq 1\), there exists \(c (h ) > 0\) such that \[\frac{1}{ X} \left\lvert \sum_{n \leq X } \lambda (n ) \lambda (n + h ) \right\rvert \leq 1 - c (h).\]

How about the \(k\)-fold divisor function in short intervals? Let \[d_k (n ) = \sum_{n = m_1 \dotsb m_k } 1.\] The asymptotic formula for \(d_k\) in short intervals is that \[\sum_{x < n \leq x + H} d_k (n) = (1 + o(1)) P_{k - 1} (\log x) H,\] where \(P_k (x)\) is an explicit polynomial of degree \(k\). Here

  • \(k = 2\) and \(x^{131/416 + \varepsilon} \leq H \leq x\) (Hulexy).

  • \(k = 3\) and \(x^{43 / 9 6 + \varepsilon} \leq H \leq x\) (Holesnik).

  • \(4 \leq k \leq 5\) and \(x^{\frac{3 k - 4}{4 k } + \varepsilon} \leq H \leq x\) (Hulexy).

  • \(k \geq 6\) and \(x^{\frac{11}{20} + \varepsilon} \leq H \leq x\) (Matomaki–Teravainen).

It is natural to ask, can we give an almost all result for divisor bounded functions (or \(d_k\) bounded multiplicative functions). Yes, but the general theorem is a bit complicated; we will try to say it at the end of the talk. Let’s start with the simple case of just the divisor function \(d_k\).

Theorem 9 (Mangerel, 2023). If \(h_0 \rightarrow \infty\) as \(X \rightarrow \infty\) and \(h \geq h_0 (\log X)^{(k - 1 )^2 }\), then \[\frac{1}{h} \sum_{x < n \leq x + h } d_k (n ) - \frac{1}{x} \sum_{x < n \leq 2 x } d_k (n) = o (\log^{k - 1} x)\] for all but \(o (X)\) values of \(x \in [X, 2 X]\).

Main results

Theorem 10 (S. 2023). Let \(X > 0\) be sufficiently large. If \(h \geq (\log X )^{(1 + \varepsilon) k \log k - k + 1}\), then \[\frac{1}{h} \sum_{x < n \leq x + h} d_k (n) - \frac{1}{x} \sum_{x < n \leq 2 x} d_k (n) = o (\log^{k - 1} x)\] for all but at most \(o(X)\) integers \(x \in [X,2 X]\).

Agniva Dasgupta, Second moment of twisted cusp forms along a coset

We’ll talk in particular about moments of \(L\)-functions. We’ll continue in particular with the theme of short intervals that we’ve already seen in this conference, but in a slightly different context.

\(L\)-functions are hard to study individually, but often we study them as a collection, via families of \(L\)-functions. Often that is does statistically, via moments of families of \(L\)-functions. Thus, let \(\mathcal{F}\) be a family of \(L\)-functions. We’re looking at expressions like \[\frac{1}{\lvert \mathcal{F} \rvert} \sum_{f \in \mathcal{F} } \left\lvert L (\tfrac{1}{2} , f) \right\rvert^k.\] More generally, one can look at expressions like \[\frac{1}{T} \int_0^T \left\lvert L (\tfrac{1}{2} + i t ) \right\rvert^k \, d t.\] The goal is to get asymptotic expansions or upper bounds for these.

The \(L\)-functions that we’ll focus on today will mostly be on \(\mathop{\mathrm{GL}}(1)\) (the zeta function or Dirichlet \(L\)-functions) or \(\mathop{\mathrm{GL}}(2)\) (\(L\)-functions associated to cusp forms, possibly twisted by \(\mathop{\mathrm{GL}}(1)\)). We’ll focus on

  • the \(4\)th moment for \(\mathop{\mathrm{GL}}(1)\), and

  • the \(2\)nd moment for \(\mathop{\mathrm{GL}}(2)\).

We consider \[\label{eqn:d1bbf1c74665}\tag{2} \frac{1}{T} \int_0^T \left\lvert \zeta (\tfrac{1}{2} + i t) \right\rvert^4 \, d t\] or, for the \(q\)-aspect, \[\label{eqn:d1bbf1c80437}\tag{3} \frac{1}{ \phi^\ast (q)} \sum_{\chi(q)} \left\lvert L (\tfrac{1}{2}, \chi ) \right\rvert^4.\] The overarching conjecture is the generalized Lindelöf hypothesis (GLH). What that says is that if you have an \(L\)-function, and let’s say we fix \(s\) at \(1/2 + i t\), then \[\left\lvert L (f, s) \right\rvert \ll \left\lvert q (f, s ) \right\rvert^\varepsilon\] for any fixed \(\varepsilon> 0\). If you’re unfamiliar with the analytic conductor, then just think of it as something that depends upon all the quantities that we use when working with \(L\)-functions, such as the arithmetic conductor \(q\) and \(s\). This implies that the moment \((2)\) is \(\operatorname{O}(T^{\varepsilon})\) and that \((3)\) is \(\operatorname{O}(q^\varepsilon)\). Such moment bounds are called Lindelöf on average.

For the zeta function, the first breakthrough (i.e., result with power-savings error term) was due to Heath–Brown (1979), who showed that \[\frac{1}{T} \int_0^T \left\lvert \zeta (\tfrac{1}{2} + i t) \right\rvert^4 \, d t = P_4 (\log T) + \operatorname{O}( T^{-1/8 + \varepsilon} ).\] The \({\mathop{\mathrm{GL}}}_2\) variant of this was done by Anton Good (1982), who showed that \[\frac{1}{T} \int_0^T \left\lvert L (\tfrac{1}{2} + i t, f) \right\rvert^2 \, d t = P_1 (\log T) + \operatorname{O}(T^{- 2/3 + \varepsilon}).\] It also follows immediately from his work that \[\int_T^{T + T^{2/3} } \left\lvert L (\tfrac{1}{2} + i t, f) \right\rvert^2 \, d t \ll T^{2/3 + \varepsilon}.\]

For the \(q\)-aspect, Young (2011) showed that \[\frac{1}{ \varphi^\ast (q)} \sum_{\chi (q)}^\ast \left\lvert L (\tfrac{1}{2}, \chi ) \right\rvert^4 = P_4 (\log Q) + \operatorname{O}(q^{- 5/512 + \varepsilon}).\] This was the first power savings estimate for the fourth moment. The \({\mathop{\mathrm{GL}}}_2\) analogue of this was done by Blomer–Fouvry–Kowalski–Michel–Milicevic–Sawin (2017), who showed that for a fixed \({\mathop{\mathrm{GL}}}_2\) form \(f\), \[\frac{1}{\varphi (q)} \sum_{\chi (q)} \left\lvert L (\tfrac{1}{2}, f \otimes \chi )\right\rvert^2 = P_1 (\log q) + \operatorname{O}(q^{- 1/144 + \varepsilon}).\]

The speaker has been working on moments over short intervals. We want to look at things like \[\int_T^{T + H } \left\lvert L (\tfrac{1}{2} + i t ) \right\rvert^k \, d t\] where \(H\) is significantly smaller than \(T\). Here GLH gives \(\operatorname{O}(H T^\varepsilon)\). One of the first significant breakthroughs here was due to Iwaniec (1978), who showed that \[\int_T^{T + T^{2/3} } \left\lvert \zeta (\tfrac{1}{2} + i t ) \right\rvert^4 \, d t \ll_\varepsilon T^{2/3 + \varepsilon}.\]

Mayank Pandey, \(L^1\)-norms of exponential sums with multiplicative coefficients

The general problem is to understand the distribution of the magnitude of exponential sums of some arithmetically interesting coefficients \(a : \mathbb{N} \rightarrow \mathbb{C}\), namely \[\left\lvert \sum_{n \leq X} a (n ) e(n \alpha) \right\rvert.\] One approach that gives a lot of information about these is through the moments \[\int_0^1 \left\lvert \sum_{n \leq X } a (n ) e (n \alpha ) \right\rvert^s \, d \alpha.\] We focus on the \(L^1\)-norm. Here there is comparatively little known. Less structure compare to the even moments.

Why would we want to understand such \(L^1\)-norms?

There’s the general problem of understanding the distribution, which one might want for its own sake. It’s not inconceivable that it might show up in applications of the circle method. Finally, there’s an interesting result of Bourgain:

Theorem 43. Any finite subset \(A \subseteq \mathbb{N}\) has a sumfree subset \(B\) with \[\lvert B \rvert \geq \frac{1}{3} \lvert A \rvert + \frac{1}{ \log \lvert A \rvert} \int_0^1 \left\lvert \sum_{n \in A } e (n \alpha ) \right\rvert \, d \alpha.\]

The best current result (also due to Bourgain) is that the biggest such \(B\) has size \[\geq \frac{1}{3} \lvert A \rvert + \frac{2}{3}.\] This improves upon a simpler result of Erdos where one just has \(|B| \geq (1/3) |A|\) under weaker conditions.

One might hope that sets with small \(L^1\)-norm can be characterized, then manually dealt with, to obtain \[\lvert B \rvert \geq \frac{1}{3} \lvert A \rvert + \psi (\lvert A \rvert),\] for some \(\psi(X) \rightarrow \infty\). Bloom has announced some results in this direction for general sets.

Let’s turn to basic considerations regarding what one can prove. It’s not hard to show that \[\int_0^1 \left\lvert \sum_{n \leq X} e (n \alpha ) \right\rvert \, d \alpha \asymp \log X.\] This is sharp, due to a (resolved) conjecture of Littlewood: \[\int_0^1 \left\lvert \sum_{n \leq X} a (n ) e (n \alpha ) \right\rvert \, d \alpha \gg \sum_{j \leq k} \frac{\left\lvert a (n_j ) \right\rvert}{j},\] where \(\{n_1, \dotsc, n_k\}\) are those \(n \leq X\) at which \(a (n) = 0\). One expects that the \(L^1\)-norm is small only if the coefficients are structured in arithmetic progressions, and the aforementioned work of Bloom makes this precise when the \(L^1\)-norm is very small.

For the case where \(a(n)\) is a multiplicative function, here are some examples of results.

Theorem 44 (Balog, Ruzsa). For fixed \(r \geq 2\), we have \[\int_0^1 \left\lvert \sum_{n \leq X} 1_{r\text{-free}}(n) e (n \alpha ) \right\rvert \, d \alpha \asymp X^{\frac{1}{r + 1}}.\]

Theorem 45 (Vaughan, Goldston). We have \[\sqrt{X} \ll \int_0^1 \left\lvert \sum_{n \leq X} \Lambda (n ) e (n \alpha ) \right\rvert \, d \alpha \leq \left( \frac{1}{\sqrt{2}} + o(1) \right) \sqrt{X \log X}.\]

Theorem 46 (Balog–Rusza). For \(a (n)\) supported on squarefree \(n\), we have \[\int_0^1 \left\lvert \sum_{n \leq X } a (n ) e (n \alpha ) \right\rvert \, d \alpha \gg X^{1/ 8 - o(1)} \left( \frac{1}{X} \sum_{n \leq X } \lvert a (n ) \rvert^2 \right)^{1/2}.\]

Theorem 47 (P.). With \(\lambda_f (n)\) the normalized Fourier coefficients of an \({\mathop{\mathrm{SL}}}_2(\mathbb{Z})\) modular form, we have \[\int_0^1 \left\lvert \sum_{n \leq X } \lambda_f (n ) e (n \alpha ) \right\rvert \, d \alpha = C X^{1/2} + \operatorname{O}(X^{1/2 - \delta })\] for some \(\delta > 0\) (probably \(\delta = 1/100\) suffices).

One of the main results that we’ll state today is the following, joint with Maksym Radziwill.

Andrei Seymour–Howell, Rigorous computation of Maass cusp forms of squarefree level

\(S_\lambda(N)\): the space of Maass cusp forms of level \(N\) (trivial character) with eigenvalue \(\lambda\).

Hecke operators \(T_n\), Fourier expansions with coefficients \[a(n) = \lambda(n), \qquad a(-n) = \varepsilon\lambda(n),\] where \(\varepsilon= \pm 1\).

History of computations. In the 1990’s, Hejhal developed an algorithm to compute Maass cusp forms, generalized by Stromberg in 2006 to work for general congruence (and non-congruence) subgroups. This argument works very well in practice, but relies on a heuristic argument, and it has not been proven yet to rigorously converge to genuine Maass cusp forms. One gets number that look like Maass forms, but one can’t tell if it is an exact Maass form.

Booker–Strombergsson–Venkatesh (2006) developed an algorithm to verify the computations of Maass cusp forms. Originally derived only for level \(1\), but recent work of Child has generalized this to general level and character.

Booker and Strombergsson (2006) used the Selberg trace formula for computations of Maass cusp forms. However, they focused mainly on proving the non-existence of Maass forms in certain cases.

The main tool we’ll use here is the Selberg trace formula. Hecke eigenbasis \(f_j\), with Laplace eigenvalues \(\lambda_j\) and Hecke eigenvalues \(a_j (n)\). Let \(H\) be a nice (analytic) test function. Then the spectral sum \[t(n,H) := \sum_j a_j (n ) H (\lambda_j )\] admits a geometric expansion.

Aside to modular forms. Explicit versions of the trace formula have been used to compute basis elements of the spaces of modular forms. Example: PARI command mfeigenbasis. Since these spaces are finite-dimensional, one can use Hecke operators and linear algebra to extract Fourier coefficients of basis elements.

In the Maass form case, our spaces become infinite-dimensional. We’ll mimic the idea, though, by taking the test function to decay rapidly enough that an essentially bounded number of eigenvalues contribute.

As for details, we define, for a sequence of real numbers \(c(m)\) (here \(1 \leq m \leq M\)) satisfying \(c(m) = 0\) if \((m,N) > 1\), \[Q (c, H) := \sum_{j = 1 }^\infty \left( \sum_{m = 1 }^M c (m ) a_j (m ) \right)^2 H (\lambda_j).\] We can rewrite this using Hecke multiplicativity.

Intuitively, suppose we have some putative numerical approximations \(\tilde{\lambda}_j\) and \(\tilde{a}_j(m)\) to \(\lambda_j\) and \(a_j(m)\). Then we want to choose numbers \(c_i(m) = 0\) if \((m, N ) > 1\) and \[\sum_{m = 1}^M c_i (m ) \tilde{a}_j(m) = \begin{cases} 1 & \text{ if } j = i, \\ 0 & \text{ otherwise.} \end{cases}\] Let \(\tilde{H}_i(\lambda) = H(\lambda) (\lambda - \tilde{\lambda}_i)^2\). For the verification, we shall prove that there exists a Laplace eigenvalue near \(\tilde{\lambda}_i\).
